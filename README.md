## Let's get warm
```bash
$ npm i -g cli-fireplace
$ cli-fireplace
```

### option
```bash
cli-fireplace [frame_rate]
```
example: `$ cli-fireplace 10`

## Too laggy?
I recommend GPU-accelerated terminal such as [alacritty](https://github.com/jwilm/alacritty)
